# Todo Web App

A simple Todo web application built with Flask.

## Features

- Add todo
- List todos
- Toggle todo status
- Delete todo
- SQLite persistence

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
